# Burger Xpress Android App

## 📱 Project Overview
Burger Xpress is a mobile application designed to make burger ordering fast, easy, and convenient. The application allows users to browse different burger options, customize ingredients, and place orders through a simple and user-friendly interface.

## 🚀 Features
- Browse different burger menus
- Customize burger ingredients
- Select spice levels and portion sizes
- View total price before confirming order
- Order tracking system
- Login and registration system
- Debit / Credit payment options
- User profile management

## 🛠 Technologies Used
- Android Studio
- Java / Kotlin
- XML Layouts
- Material UI Design

## 📷 Screens
- Splash Screen
- Onboarding Screens
- Login / Register
- Home Page
- Burger Customization
- Order and Delivery
- Payment Success
